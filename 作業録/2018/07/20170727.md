	- 改行コードが混ざるので注意
		- テキストエディターの設定で、改行コードをLFにするのが安定しそう
		- [参考：気をつけて！Git for Windowsにおける改行コード](https://qiita.com/uggds/items/00a1974ec4f115616580)
	- Gitのautocrlfを変えておいたほうが良さげ
		- git config --global autocrlf false
		- winだけでやるときはtrueで良さげ
